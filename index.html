Genetic LIMS Demo is ready

The app source is built in this folder. To run it, use:

python3 download_dataset.py
npm install
npm start

Then open:

http://localhost:3000

Note: This environment does not currently include npm or node, so the server cannot be started here.

Files created:

  • server.js — Express backend with SQLite support
  • public/index.html, public/styles.css, public/app.js — frontend UI
  • download_dataset.py — downloads a sample genetic dataset
  • data/samples.csv — downloaded sample metadata
